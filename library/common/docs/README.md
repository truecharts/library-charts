# Root Chart Context

| Key              |   Type   | Required | Helm Template | Default | Description                                 |
| :--------------- | :------: | :------: | :-----------: | :-----: | :------------------------------------------ |
| .Values.TIMEZONE | `string` |    ✅    |      ❌       |  `UTC`  | Timezone that is used everywhere applicable |

---

Examples:

```yaml
TIMEZONE: Europe/Berlin
```
